# Skill Factory

Skill Factory helps you create new `xx-factor-*` Skills with a consistent structure.

## Generate with script

```bash
python scripts/generate_factor_skill.py xx-factor-cli --title "xx-factor-cli" --category cli
```

## Manual template

Use `skill-factory/xx-factor-skill-template/` if you prefer to copy files manually.

Every generated Skill should include:

- `SKILL.md`
- `checklist.md`
- `scoring-rubric.md`
- `audit-report-template.md`
- `refactor-plan-template.md`
- `usage-examples.md`
- `skill.json`
