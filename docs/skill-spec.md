# Factor Skill Spec

A Factor Skill is a repeatable project optimization unit.

## Required files

- `SKILL.md`: routing and execution instructions
- `checklist.md`: audit checklist
- `scoring-rubric.md`: quality scoring guide
- `audit-report-template.md`: output report shape
- `refactor-plan-template.md`: change plan shape
- `usage-examples.md`: example prompts

## Required sections in SKILL.md

- Name
- Purpose
- When to use
- When not to use
- Inputs
- Workflow
- Output format
- Review dimensions
- Refactor priorities
- Examples
- Version

## Scoring

Use 0-10 scores per dimension, with short evidence notes. Avoid vague praise. Prefer evidence from actual files.
