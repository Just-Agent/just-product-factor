# Complete Product Audit Prompt

```text
Use product-factor-skills to audit this repository.

First identify which Skill is most relevant. If multiple Skills apply, choose the primary one and mention secondary Skills.

Read the project README, configuration, source structure, examples, scripts, docs, workflows, changelog, and logs.

Return:
1. A one-page project diagnosis.
2. A factor scorecard.
3. The top release blockers.
4. A prioritized refactor plan.
5. Validation commands to run.
6. A next-version roadmap.

Do not only write advice. If you can modify files, make concrete improvements and update logs.
```
