# CLI Audit Prompt

```text
Use the 11-factor-cli skill to audit this CLI for installability, command UX, configuration, error handling, logging, testing, packaging, and release readiness.

Prioritize:
1. A clean install path.
2. Useful root and subcommand help.
3. A working happy path.
4. Clear error messages.
5. Scriptable output and stable exit codes.
6. Smoke tests and release notes.

Return an audit report, score table, refactor plan, validation commands, and known limits.
```
