# Base Factor Skill Checklist

Use this checklist when creating or reviewing any Factor Skill.

- [ ] The Skill has a clear project domain.
- [ ] The Skill explains when to use it.
- [ ] The Skill explains when not to use it.
- [ ] The Skill has explicit expected inputs.
- [ ] The Skill has explicit expected outputs.
- [ ] The Skill has a checklist.
- [ ] The Skill has a scoring rubric.
- [ ] The Skill has audit and refactor templates.
- [ ] The Skill has usage examples.
- [ ] The Skill has a `skill.json` manifest.
- [ ] The Skill version matches root `VERSION`.
- [ ] The Skill is listed in `product-factor-skills.json`.
- [ ] The Skill can be validated by `scripts/scan_factor_project.py`.
