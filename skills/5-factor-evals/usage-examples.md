# 5-factor-evals Usage Examples

## Agent project

```text
Use the 5-factor-evals skill to audit this Agent project for golden cases, regression tests, failure taxonomy, and release gates.
```

## RAG project

```text
Use the 5-factor-evals skill as a secondary review after 9-factor-rag. Focus on retrieval regression cases, citation correctness, and answer quality gates.
```

## Product release

```text
Use the 5-factor-evals skill to decide what validation evidence this repository needs before v1.0.0.
```

## HTML or generated UI project

```text
Use the 5-factor-evals skill to define screenshot, accessibility, and content regression checks for generated HTML pages.
```
