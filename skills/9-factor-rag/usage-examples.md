# 9-factor-rag Usage Examples

## Audit prompt

```text
Use the 9-factor-rag skill to audit this project.
Inspect the available files, apply the checklist, score the project, and return an audit report with prioritized fixes.
```

## Refactor prompt

```text
Use the 9-factor-rag skill to refactor this project toward release readiness.
Make concrete file changes where possible, validate the result, and update logs or release notes.
```

## First-version prompt

```text
There is no previous version. Use the 9-factor-rag skill to create a first usable project structure, including README, examples, validation script, logs, and release notes.
```
