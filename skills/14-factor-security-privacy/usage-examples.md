# 14-factor-security-privacy Usage Examples

## Product release safety

```text
Use the 14-factor-security-privacy skill to audit this repository for secrets, permissions, data handling, dependency hygiene, CI/CD safety, and release blockers.
```

## Agent tool project

```text
Use the 14-factor-security-privacy skill after 12-factor-agents. Focus on tool permissions, prompt injection risk, destructive actions, secrets, and log safety.
```

## MCP server

```text
Use the 14-factor-security-privacy skill with 7-factor-mcp to review this MCP server for schemas, permissions, side effects, and data exposure risks.
```

## RAG product

```text
Use the 14-factor-security-privacy skill with 9-factor-rag to audit private/public source boundaries, citation leakage, retention, and generated answer safety.
```

## GitHub Actions

```text
Use the 14-factor-security-privacy skill with 8-factor-github-actions to check workflow permissions, secrets exposure, release artifact safety, and deployment approval gates.
```
