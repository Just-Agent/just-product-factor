# 11-factor-cli Usage Examples

## Basic audit

```text
Use the 11-factor-cli skill to audit this CLI for installability, command UX, configuration, error handling, logging, testing, packaging, and release readiness.
```

## Refactor request

```text
Use the 11-factor-cli skill to turn this prototype command-line tool into a release-ready CLI. Prioritize install, help output, happy path, errors, JSON output, smoke tests, and release notes.
```

## Combined review

```text
Use the 12-factor-app skill for the overall product architecture, then use the 11-factor-cli skill for the command interface and packaging quality.
```

## Expected output

The Agent should return:

- CLI audit report
- Score table
- Command UX issues
- Refactor plan
- Validation commands
- Release readiness decision
